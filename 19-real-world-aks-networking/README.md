## 🎯 핵심

- AKS 아키텍처: kubelet, CNI, kube-proxy 통합
- Azure VNet과 Kubernetes 네트워크의 통합
- 고급 시나리오: Service Mesh, Multi-cluster, Hybrid Networking
- 성능 튜닝과 보안 강화
- 엔터프라이즈 배포 패턴

## 실습 사례

1. 새로운 AKS 클러스터에 Cilium CNI 설치
2. 멀티 테넌트 격리: NetworkPolicy로 namespace 간 통신 제어
3. 마이크로서비스 아키텍처에서의 패킷 흐름 분석
4. 성능 문제 진단: 지연시간, 패킷 손실 원인 분석
5. 보안 감사: 정책 준수 확인 및 위반 탐지
